Для оценки дана функция:

int calc(int* arr, int size)
{
  int ans = 0;
  for (int i = 1; i < size; i++)
  {
    ans += arr[i] - arr[i-1];
  }
  return ans;
}

Функция принимает два параметра, массив int* arr по указателю, и переменную int size
по значению. Также в теле функции создаётся две переменные int ans, int i. 
Отсюда значит, что общий размер функции составляет примерно 3 * int байт (int size, ans, i),
так как массив int* arr принимается по указателю его размер не влияет на размер функции.
Объём памяти занимаемый функцией составляет O(1).
В функции используется один цикл, который один раз последовательно перебирает все элементы 
массива arr. Время работы функции = O(n).
